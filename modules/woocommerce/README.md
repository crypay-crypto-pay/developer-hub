# Crypay WooCommerce Plugin

> **Coming in Phase 3.**

A WordPress/WooCommerce payment gateway plugin that integrates Crypay's API will be developed here.

## Planned Features

- Payment gateway registration in WooCommerce checkout
- API key configuration in WooCommerce admin
- Redirect to Crypay hosted payment page
- Webhook receiver for automatic order status updates (`Processing` on `SUCCESS`)
- Test mode support

## Target Compatibility

- WordPress 6.x
- WooCommerce 8.x
- PHP 8.1+
